# Project 2: Iris Flower Classification

## Objective

Build a machine learning model that classifies Iris flowers into different species using a Decision Tree Classifier.

## Dataset

The Iris dataset contains measurements of flower characteristics:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

### Target Classes

* Setosa
* Versicolor
* Virginica

## Machine Learning Workflow

1. Load Dataset
2. Explore Dataset
3. Split Training and Testing Data
4. Train Decision Tree Model
5. Generate Predictions
6. Evaluate Model Performance
7. Predict Custom Samples

## Evaluation Metrics

* Accuracy Score
* Classification Report
* Confusion Matrix

## Concepts Used

* Supervised Learning
* Classification
* Decision Trees
* Model Evaluation

## Run

```bash
python Project2.py
```

# Project 4: Image Recognition Using Deep Learning

## Objective

Implement image recognition using a pre-trained deep learning model.

## Model Used

**ResNet18** (Pre-trained on ImageNet)

## Workflow

1. Load Pre-trained Model
2. Load Input Image
3. Preprocess Image
4. Perform Prediction
5. Display Recognized Object

## Features

* Uses a pre-trained Convolutional Neural Network (CNN)
* Supports object recognition from images
* No model training required
* Fast and accurate predictions

## Example

### Input

An image of a Doberman dog.

### Output

```text
===== Image Recognition Result =====
Predicted Object: Doberman
```

## Concepts Used

* Computer Vision
* Deep Learning
* Transfer Learning
* Neural Network Inference

## Run

```bash
python Project4.py
```
